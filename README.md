# ProductCompare

เว็บเปรียบเทียบราคาสินค้าสำหรับใช้ตอนเดินซื้อของ กรอกชื่อสินค้า ราคา และจำนวน เพื่อดูว่ารายการไหนคุ้มที่สุดแบบรวดเร็วบนมือถือและเดสก์ท็อป

## ฟีเจอร์

- UI ใช้ Bootstrap 5.3 ร่วมกับ custom CSS
- รองรับมือถือและเดสก์ท็อป
- ฟอร์มกรอกข้อมูลแบบแถวเดียวบนจอปกติ และปรับเป็นคอลัมน์เดียวบนจอเล็ก
- คำนวณราคาต่อหน่วยอัตโนมัติ
- ตารางผลลัพธ์เรียงจากคุ้มที่สุดไปแพงที่สุด
- ไฮไลต์ตัวเลือกที่คุ้มที่สุดด้วยข้อความ `⭐ คุ้มสุด`
- ค้นหารายการตามชื่อสินค้า และลบรายการได้
- บันทึกข้อมูลไว้ใน browser ด้วย Local Storage
- มีปุ่มสลับโหมดมืด/สว่าง และจดจำค่าธีมที่เลือกไว้
- ไม่ใช้ cache จาก Service Worker แล้ว เพื่อให้โหลดไฟล์ล่าสุดเสมอ

## การใช้งานในเครื่อง

เปิด `index.html` ด้วยเบราว์เซอร์ได้เลย หรือใช้ Live Server ใน VS Code

## พฤติกรรมปัจจุบัน

- ข้อมูลรายการสินค้าเก็บใน Local Storage ของเบราว์เซอร์
- ธีมมืด/สว่างเก็บใน Local Storage ของเบราว์เซอร์
- Service Worker ถูกปิดการใช้งาน และมีโค้ดล้าง cache เก่าอัตโนมัติ
- แอปจึงไม่รองรับ offline cache แบบเดิมแล้ว

## วิธีใช้งาน

1. กรอกชื่อสินค้า ราคา และจำนวน
2. กด `เพิ่มรายการ`
3. ตารางจะแสดงราคาต่อหน่วยและเรียงรายการจากถูกต่อหน่วยที่สุดขึ้นก่อน
4. แถวที่คุ้มที่สุดจะขึ้น `⭐ คุ้มสุด`
5. ใช้ช่องค้นหาเพื่อกรองรายการตามชื่อสินค้า
6. ใช้ปุ่ม `ลบ` เพื่อลบรายการ หรือ `ล้างทั้งหมด` เพื่อล้างทั้งตาราง

## Deploy ขึ้น GitHub Pages (ผ่าน GitHub Actions)

1. สร้าง repository ใหม่บน GitHub แล้ว push โค้ดขึ้น branch `main`
2. ใน GitHub ไปที่ `Settings > Pages`
3. ที่หัวข้อ `Build and deployment` ให้เลือก `Source: GitHub Actions`
4. push ครั้งถัดไป (หรือกด Run workflow ที่หน้า Actions) ระบบจะ deploy ให้อัตโนมัติ
5. รอ workflow สำเร็จ แล้วเข้า URL ที่ GitHub Pages แจ้ง

## โครงสร้างไฟล์

- `index.html` หน้าเว็บหลักและโครงสร้าง UI
- `style.css` สไตล์, layout, dark mode, และ responsive behavior
- `app.js` ตรรกะคำนวณ, render ตาราง, Local Storage, และ theme toggle
- `manifest.webmanifest` metadata ของเว็บแอป
- `service-worker.js` ไฟล์ cleanup สำหรับยกเลิก cache เก่า
- `.github/workflows/deploy.yml` workflow สำหรับ deploy ขึ้น GitHub Pages
