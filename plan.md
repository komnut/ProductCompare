# ProductCompare Plan

## Should Have (MVP)

- Add product name, price, and quantity quickly
- [x] Auto-calculate price per unit accurately
- Sort results from cheapest per unit to most expensive
- [x] Highlight best option clearly
- Edit and delete each item
- [x] Validate inputs and show friendly errors
- Persist data in local storage so refresh does not lose entries
- Mobile-first layout for in-store usage
- Fast load and responsive interactions

## Nice to Have

- Unit presets and conversion (ml, L, g, kg, pcs)
- Multi-pack support (for example 3 x 250ml)
- Discount and promo calculator (percent off, buy X get Y)
- Scan barcode to prefill product info
- [x] Share comparison as image or link
- Import/export data (CSV or JSON)
- Price history and trend charts
- Category tagging and filtering
- Offline-first PWA behavior with install prompt
- Multi-language UI and accessibility improvements
